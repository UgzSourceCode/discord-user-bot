# discord-record-bot
## Description
Basic bot for recording meetings

## Commands
 - "npm start" - run application
 - "npm run dev" - run application in watch mode
 - "docker build -t discord-bot ." - build docket image

## Modules
 - discord.js
 - dotenv
Dev:
 - @types/node
 - ts-node
 - ts-node-dev
 - typescript
